# Pick a card

A simple console program for choosing a random card. Written in [Ruby](https://en.wikipedia.org/wiki/Ruby_(programming_language))

#### Usage:

Clone repo
```
$ git clone git@github.com:phobco/pick-a-card.git
```

Enter the folder
```
$ cd pick-a-card/
```

Run
```
$ ruby pick_a_card.rb
```
